# 로또 번호 추첨 프로그램

## 개요

이 애플리케이션은 사용자에게 로또 번호를 추천해주는 기능을 제공합니다. Web Component를 사용하여 캡슐화되고 재사용 가능한 UI를 생성합니다.

## 기능

-   **번호 생성:** "번호 생성" 버튼을 클릭하면, 1부터 45까지의 숫자 중에서 무작위로 6개의 숫자를 선택하여 애니메이션과 함께 화면에 표시합니다.
-   **디자인:**
    -   현대적이고 대담한 디자인 가이드라인을 따릅니다.
    -   배경에 미묘한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
    -   로또 추첨기 카드에 다중 레이어 그림자를 적용하여 깊이감을 줍니다.
    -   생성된 각 번호에 구간별로 다른 색상 그라데이션을 적용합니다.
    -   버튼에는 상호작용 시 '빛나는' 효과가 있습니다.
-   **구조:**
    -   핵심 기능은 `<lotto-drawing-machine>` Web Component로 캡슐화됩니다.
    -   Shadow DOM을 사용하여 스타일과 동작이 외부와 충돌하지 않도록 합니다.

## 현재 요청: UI 개선 및 Web Component로 전환

**실행 계획:**

1.  **`blueprint.md` 업데이트:** 현재 상태와 개선 계획을 반영하여 문서를 업데이트합니다.
2.  **`index.html` 수정:** 기존 `div` 구조를 제거하고 `<lotto-drawing-machine></lotto-drawing-machine>` 단일 커스텀 엘리먼트를 사용하도록 변경합니다.
3.  **`style.css` 수정:** 전역 스타일을 정리하고, 바디에 노이즈 텍스처 배경을 추가합니다. 컴포넌트 관련 스타일은 `main.js`로 이동합니다.
4.  **`main.js` 수정:**
    -   `HTMLElement`를 상속받는 `LottoDrawingMachine` 클래스를 생성합니다.
    -   컴포넌트의 HTML 구조, 스타일, 그리고 로또 번호 생성 로직을 클래스 내부에 구현합니다.
    -   Shadow DOM을 사용하여 컴포넌트를 캡슐화합니다.
    -   `customElements.define`을 사용하여 커스텀 엘리먼트를 등록합니다.
    -   스타일은 생동감 있는 색상, 그림자, 애니메이션을 포함하여 대담한 디자인 가이드라인을 따릅니다.
